## 📋 Visão Geral

Uma coleção de ferramentas web gratuitas e acessíveis para uso no dia a dia. Este projeto consiste em um site intuitivo com uma página inicial e cinco aplicações úteis, cada uma projetada para resolver problemas específicos.

## 🧰 Ferramentas Disponíveis

### 🎲 Gerador de Números Aleatórios
Gere números aleatórios rapidamente dentro de intervalos personalizados.

### 🔐 Gerador de Senhas Aleatórias
Crie senhas seguras e aleatórias com opções de complexidade personalizáveis.

### 🔄 Conversor de Texto
Transforme facilmente textos entre maiúsculas e minúsculas.

### 📊 Contador de Palavras e Caracteres
Analise seu texto com estatísticas detalhadas de palavras e caracteres.

### 🔍 Comparador de Texto
Compare dois textos e identifique as diferenças entre eles.

## 🚀 Como Rodar Localmente

### Pré-requisitos

- Node.js (versão recomendada: 14.x ou superior)
- npm ou yarn

### Instalação

Clone este repositório:

```
git clone https://github.com/WillianDDaniel/online-web-tools
```

Acesse a pasta do projeto:

```
cd online-web-tools
```

Instale as dependências:

```
npm install
```

ou

```
yarn
```

Inicie o servidor de desenvolvimento:

```
npm run dev
```

ou

```
yarn dev
```

Abra no navegador:

```
http://localhost:5173/online-web-tools/
```

## 📦 Build para Produção

Para gerar uma versão otimizada para produção:

```
npm run build
```

ou

```
yarn build
```

Os arquivos finais estarão disponíveis na pasta `dist/`.
