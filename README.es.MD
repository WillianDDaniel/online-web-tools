## 📋 Descripción

Una colección de herramientas web gratuitas y accesibles para el uso diario. Este proyecto consiste en un sitio web intuitivo con una página principal y cinco aplicaciones útiles, cada una diseñada para resolver problemas específicos.

## 🧰 Herramientas Disponibles

### 🎲 Generador de Números Aleatorios
Genera números aleatorios rápidamente dentro de rangos personalizados.

### 🔐 Generador de Contraseñas Aleatorias
Crea contraseñas seguras y aleatorias con opciones de complejidad personalizables.

### 🔄 Convertidor de Texto
Transforma fácilmente texto entre mayúsculas y minúsculas.

### 📊 Contador de Palabras y Caracteres
Analiza tu texto con estadísticas detalladas de palabras y caracteres.

### 🔍 Comparador de Texto
Compara dos textos e identifica las diferencias entre ellos.

## 🚀 Cómo Ejecutar Localmente

### Requisitos Previos

- Node.js (versión recomendada: 14.x o superior)
- npm o yarn

### Instalación

Clona este repositorio:

```
git clone https://github.com/WillianDDaniel/online-web-tools
```

Accede al directorio del proyecto:

```
cd online-web-tools
```

Instala las dependencias:

```
npm install
```

o

```
yarn
```

Inicia el servidor de desarrollo:

```
npm run dev
```

o

```
yarn dev
```

Abre en el navegador:

```
http://localhost:5173/online-web-tools/
```

## 📦 Build para Producción

Para generar una versión optimizada:

```
npm run build
```

o

```
yarn build
```

Los archivos estarán disponibles en la carpeta `dist/`.
